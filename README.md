# FoxBotGamer
FoxBot Gamer just a little game app is building by `ElectronJS ~ A NodeJS Framework`.

- You can install the `latest version (v1.0)` [here](https://github.com/KaizFox/FoxBotGamer/releases/tag/1.1.0).
